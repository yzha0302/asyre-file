# Changelog

## [1.0.0] - 2026-04-05

### Added
- Single-file Python web server with inline HTML/CSS/JS
- CodeMirror editor with 20+ language modes
- Live Markdown preview with Mermaid, math, code highlighting
- Multi-user authentication (admin/editor/viewer roles)
- Path-based permission system
- File tree with colored SVG icons, drag-and-drop moving
- Right-click context menu (open, rename, copy path, copy link, download, select, delete)
- Multi-select with Cmd/Ctrl+Click, batch delete/share/copy
- File upload (button, drag-and-drop, right-click "Upload here")
- Share links (read-only or editable, single file or folder)
- PDF and Word export with custom signatures and themes
- Annotation system for review workflows
- AI editing assistant (Anthropic/OpenAI)
- Dark and light theme switching
- Recent files list
- File statistics (word count, char count, reading time)
- Activity logging with admin panel viewer
- Toast notifications
- First-run setup wizard (web UI + CLI)
- Configuration system (JSON + environment variable override)
- Agent REST API with Bearer token authentication
- Docker support
- One-line install script
